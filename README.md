# Trademark Greetings — online store assets

Product images and catalog data for the Trademark Greetings WooCommerce store.

- `assets/cards/` — 95 greeting cards, each `<slug>-front.webp` (cover) + `<slug>-inside.webp` (message)
- `assets/notepads/` — 10 die-cut notepad designs
- `data/products.json` — full catalog (title, category, SKU, image URLs, price)
- `data/woocommerce-products.csv` — ready-to-import WooCommerce product CSV

Built by BH Web Solutions.
